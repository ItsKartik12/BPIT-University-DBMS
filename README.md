# 🎓 BPIT University Management System

A full-stack Database Management System (DBMS) designed for Bhagwan Parshuram Institute of Technology (BPIT). This project transforms a theoretical Entity-Relationship (ER) model into a functional web application.

## 🚀 Features
* **Full CRUD Operations**: Add, View, and Delete records across all university entities in real-time.
* [cite_start]**Relational Mapping**: Implements 8 interconnected tables derived from complex ER relationships (1:N and M:N) [cite: 85-107].
* **Dynamic Dashboard**: A responsive frontend that automatically adapts its interface based on the selected database table.
* [cite_start]**Data Integrity**: Enforces strict relational constraints using Primary Keys and Foreign Keys [cite: 108-176].

## 🏛️ Database Schema
The system manages the following core entities as defined in the project documentation:
1. [cite_start]**Colleges**: Institutional details[cite: 91, 200].
2. [cite_start]**Departments**: Academic units managed by HODs[cite: 92, 202].
3. [cite_start]**Faculty**: Teacher profiles and departmental affiliations[cite: 94, 210].
4. [cite_start]**Students**: Enrollment and contact data[cite: 95, 204].
5. [cite_start]**Courses**: Subject catalogs offered by departments[cite: 93, 208].
6. [cite_start]**Progress Reports**: Academic performance tracking[cite: 103, 206].
7. [cite_start]**Junction Tables**: `TEACHES` and `ENROLLED` tables to handle Many-to-Many relationships [cite: 104-107].
